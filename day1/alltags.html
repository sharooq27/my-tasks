HTML TAGS

A
  • anchor tag---------> It is termed as anchor tag and it creates a hyperlink or link.

  • abbreviation tag-------> It defines an abbreviation for a phrase or longer word.

  • defines acronym for a word. (Not supported in HTML5)

  • It is used to embed sound content in HTML document.

  • address tag-------> It defines the author's contact information of the HTML article.

  • area tag--------> It defines the area of an image map.

  • article---------> It defines the self-contained content.

B
  • bold tag------->It is used to make a text bold.

  • base tag------->This tag defines the base URL for all relative URL within the document.

  • basefont tag-------->This tag is used to set default font, size and color for all elements of document. (Not supported in HTML5)

  • bdi tag--------> This tag is used to provide isolation for that part of text which may be formatted in different directions from its surrounding text.

  • bdo tag--------->It is used to override the current text direction.

  • big tag-------->This tag is used to make font size one level larger than its surrounding content. (Not supported in HTML5)

  • blockquote tag--------> It is used to define a content which is taken from another source.

  • button tag--------> It is used to represent a clickable button.

C
  • canvas tag---- It is used to provide a graphics space within a web document.

  • caption tag------> It is used to define a caption for a table.

  • center tag------> It is used to align the content in center. (Not supported in HTML5)

  • cite tag-------> It is used to define the title of the work, book, website, etc.

  • code tag-------> It is used to display a part of programming code in an HTML document.

  • col tag-------> It defines a column within a table which represent common properties of columns and used with the element.

  • colgroup tag-------> It is used to define group of columns in a table.

D
  • data tag-----> It is used to link the content with the machine-readable translation.

  • datalist------> It is used to provide a predefined list for input option.

  • dd tag-------> It is used to provide definition/description of a term in description list.

  • del tag------> It defines a text which has been deleted from the document.

  • details tag-------> It defines additional details which user can either view or hide.

  • dfn tag--------> It is used to indicate a term which is defined within a sentence/phrase.

  • dialog tag----- It defines a dialog box or other interactive components.

  • div tag-------> It defines a division or section within HTML document.

E
  • em tag-----> It is used to emphasis the content applied within this element.

  • embed tag------> It is used as embedded container for external file/application/media, etc.

F
  • fieldset tag------> It is used to group related elements/labels within a web form.

  • figcaption tag-------> It is used to add a caption or explanation for the element.

  • figure tag------> It is used to define the self-contained content, and s mostly refer as single unit.

  • font tag-------> It defines the font, size, color, and face for the content. (Not supported in HTML5)

  • footer tag------> It defines the footer section of a webpage.

  • form tag------> It is used to define an HTML form.

H
  • h1 tag-----> It defines headings for an HTML document from level 1

  • h2 tag------> It defines headings for an HTML document from level 2

  • h3 tag------->It defines headings for an HTML document from level 3

  • h4 tag-------->It defines headings for an HTML document from level 4

  • h5 tag-------->It defines headings for an HTML document from level 5

  • h6 tag-------->It defines headings for an HTML document from level 6

  • head tag------>It defines the head section of an HTML document.

  • header tag------->It defines the header of a section or webpage.

I
  • i tag------>It is used to represent a text in some different voice.

  • It defines an inline frame which can embed other content

  • img tag-----> It is used to insert an image within an HTML document.

  • ins tag-------->It represent text that has been inserted within an HTML document.

K
  • kbd tag----->It is used to define keyboard input.

L
  • list tag-------> It is used to represent items in list.
